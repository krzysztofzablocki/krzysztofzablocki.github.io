Blog Speaking Sourcery Pro



Sourcery Pro is a Mac App that extends Xcode with the ability to create your own live templates that understand your code structure.

You can create any template you want but the app also comes bundled with many popular templates to get you started.

 

 





Default templates include:

  • Mocks from any protocol
  • Codable conformance for enums with associated values
  • SwiftUI from data structure
  • Lenses / Prisms
  • Automatic interfaces from types
  • Type erasure
  • And many more…
