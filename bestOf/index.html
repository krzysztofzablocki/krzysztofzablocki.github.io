The best of my creations

This is an list of what I think are best things you should check out, a lot of it is based on usefulness / popularity but also my personal view how important or helpful they are.

Libraries / Tools

  • KZPlaygrounds - Playgrounds that can support both Swift and Objective-C + a lot of extra features that will make your life easier.
  • KZPropertyMapper - If you have to map JSON in Objective-C, this is really helpful and it requires nothing at all from your code base, no subclassing, no dependencies and no restrictions.
  • KZBootstrap - A iOS project bootstrap that’s setup for high quality coding, featuring extra scripts for safety, icon versioning and multiple environments.

Talks & Articles

  • How do I Work? - Describes my approach to working with clients, coordinating teams and establishing good practices.
  • iOS Developer Tools - A very popular talk and article I did for AltTechTalks, describing a lot of my day to day tools that help me save hours of development time.
  • Drawing smooth lines inspired by Paper - A very in-depth article when I came up with an end-to-end solution for drawing beautiful lines with anti-aliasing, I wrote it because at that time there were no tutorials how to create something like this.
  • Expanded use of Asserts - Why normal assertions are bad, and how I made a set of advanced asserts that can simplify your code and give you more safety.
  • Programmers bad habits - Talking about some things that become a habbit for developers but are considered bad.
  • Subjective Guide to Writing iOS Apps - 2 Articles about iOS app self-resolving iOS app Architecture
